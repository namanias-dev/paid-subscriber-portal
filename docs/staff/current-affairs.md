# Current Affairs

**Menu:** `Current Affairs` (group)  ·  **Permission:** Manage Current Affairs (all CA pages).

The Current Affairs section manages daily articles, compilation PDFs, categories, tags, and CA analytics.

## Articles
**Web address:** `/admin/current-affairs` — heading `Current Affairs — Articles`.
- Search/filter the list; select rows for bulk `Publish`, `Archive`, `Unpublish`, `Clear`.
- Per row: `Edit`, `Preview`, `Duplicate`, `Delete`.
- Quick links: `Categories`, `Tags`, `PDF Library`, `Daily`, `Analytics`.

### Create / edit an article
**`+ New Article`** (or `Edit`) → tabs `Basic`, `Content`, `UPSC Relevance`, `Media / PDF`, `SEO`, `Related & Promote`, `Publish Settings`. Key fields: `Title`, `Slug (URL)`, `Article type`, `Summary`, `Category`, `Current affairs date`, `Body`, `Status`, `Publish date & time`, and flags `Important`, `Trending`, `Show on homepage`, `In daily digest`, `In monthly compilation`. Save with `Create article` / `Save changes`.

## Daily Publishing
**Web address:** `/admin/current-affairs/daily` — articles grouped by current-affairs date, with published/total counts. `+ New Article`; per article `Edit` / `Preview`.

## PDF Library
**Web address:** `/admin/current-affairs/pdfs` — upload daily/monthly/general compilation PDFs. Fields: `Title`, `Kind` (`daily`/`monthly`/`general`), `Date ref`, `Upload PDF` (or URL), `Requires login`, `Requires lead (phone)`. `Add to library`; `Open` / `Delete`.

## Categories & Tags
- `/admin/current-affairs/categories` — `Name`, `Description (optional)`, `Add` / `Delete`.
- `/admin/current-affairs/tags` — `Name`, `Add` / `Delete`.

## CA Analytics
**Web address:** `/admin/current-affairs/analytics` — KPIs: `Articles`, `Total views`, `PDF downloads`, `Leads captured`, `CTA clicks`, `Quiz clicks`; plus `Most read`, `Most downloaded`, `Category performance`.

> Note: Current Affairs leads are tracked separately and do **not** appear in the Lead CRM page.
