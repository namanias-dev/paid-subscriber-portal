# Website & Settings

These pages control the **public website** content and the academy's brand/contact details. **Permission:** Manage settings (for all of these). Changes save live to the public site.

## Settings (brand & contact)
**Menu:** `People` → `Settings`  ·  **Web address:** `/admin/settings` — heading `Settings`, subtitle `Edit brand, contact details, WhatsApp & map — saved live.`
Fields: `Academy name (full)`, `Short name`, `Tagline`, `Support phone`, `WhatsApp number`, `Support email`, `Address`, `Google Maps link`, `Google Maps embed URL`, `Instagram`, `YouTube`, `Telegram`. Also shows read-only integration status (`Mode`, `Razorpay payments`, `Email (Resend)`). Save with `Save Settings`.

## Home Page
**Menu:** `Overview` → `Home Page`  ·  **Web address:** `/admin/home` — heading `Home Page`.
Tabs: `Branding / Logo`, `Hero`, `Lead Popup`, `Sections`, `Quizzes`. Controls the logo, hero headline/subheading/portrait, the timed lead popup (`Delay (seconds)`, course-interest options), section headings, the ₹50 promo band, and quiz lead-gate. Save with `Save Home Page`.

## Toppers / Results
**Menu:** `Website` → `Toppers / Results`  ·  **Web address:** `/admin/toppers` — heading `Toppers / Results`.
Add/reorder success cards. Per topper: `Rank *`, `Name *`, `Exam`, `Year`, `Order`, plus photo `Upload`/`Replace`/`Remove`. `+ Add topper`, `↑`/`↓`, `Delete`, then `Save Toppers`.

## Navigation / Header
**Menu:** `Website` → `Navigation / Header`  ·  **Web address:** `/admin/navigation` — heading `Navigation / Header`.
Toggle each public header tab `On`/`Off` and set its order. Hiding a tab only removes the menu link (direct URLs still work). Save with `Save Navigation`.

## About Page
**Menu:** `Website` → `About Page`  ·  **Web address:** `/admin/about` — heading `About Page`.
Edit the public About page text (hero, mentor section, values). Save with `Save About Page`.
