# Marketing Tools

These `Sales`-group pages help with outreach and capturing leads. **Permission:** Manage students & leads.

## Marketing / Broadcast
**Web address:** `/admin/marketing` — heading `Marketing / Broadcast`, subtitle `Templates, audiences & bulk outreach`.
- Build a lead audience using chips: `All leads`, `Warm & Interested`, `Demo attended`, `Admitted students`, `Webinar registrants`.
- KPI cards: `Total audience`, `Selected`, `Templates`, `Channels`.
- **`⬇ Export audience CSV`** to download contacts. Per template, **`Copy`** the pre-written WhatsApp message (`Webinar reminder`, `Demo follow-up`, `Fee reminder`, `Old lead reactivation`).
> ⚠️ This page copies WhatsApp text and exports CSVs; it does not send SMS. For SMS use SMS Mission Control.

## Landing Pages
**Web address:** `/admin/registrations` — heading `Landing Pages`, subtitle `Auto-generated public pages — copy & share, registrants flow into CRM`.
- A directory of shareable public links for webinars, published courses, and fixed lead funnels (e.g. `Free Counselling`, `1-Week Demo`).
- Per card: **`Copy link`**, **`Open ↗`**.

## Lead Forms Builder
**Web address:** `/admin/forms` — heading `Lead Forms Builder`, subtitle `Custom capture forms — submissions flow into the CRM`.
- **`+ New Form`** → set `Form name`, `Campaign / destination tag`, and tick `Fields` (`name`, `phone`, `city`, `state`, `course_interest`, `target_year`, `mode_pref`).
- Each form gets a shareable link (**`Copy link`**). The table shows `Form`, `Campaign`, `Fields`, `Submissions`, `Share`.

## Referrals
**Web address:** `/admin/referrals` — heading `Referrals`.
- Tracks referrer/referee pairs, admission status, and payout tier.
- When a referee is admitted and the payout is pending, **`Mark paid`** records the payout. Columns: `Referrer`, `Phone`, `Referee`, `Tier`, `Admitted`, `Payout`.
